1 GUI

Die Darstellung des Spiels wird durch die Klasse GUI ermöglicht. Für ein visuell ansprechendes Spiel ist es ratsam sich im Vorhinein Gedanken zu einer Farbpallete zu machen. Die ausgewählten Hex-Codes können Sie, dann als Konstanten speichern.

Erstellen Sie die Klasse GUI und packen Sie diese im package unotempel. 

1.1 Konstanten für die Darstellung des Spiels 

Definieren Sie folgende Konstanten für die Darstellung des Spiels:

  • die Farbe des Spielfelds 
  • die Größe des Spielfelds 
  • die Farben für die Darstellung der Karten 
  • ein Rechteck für die Darstellung der Lebenspunkte 

In dem unten stehenden Code-Block finden Sie die Werte, die wir für die Darstelung des Spiels benutzt haben. Sie können diese so übernehmen oder nach persönlichen Vorlieben anpassen. 

    // Farbe des Spielfelds
    public final static String farbeSpielfeld = "#9CAF88";
    // Maße des Spielfelds
    public final static int breite = 850;
    public final static int hoehe = 470;
    // Farben für die Darstellung der Karten
    public final static String farbeGruen = "#465726";
    public final static String farbeRot = "#F57461";
    public final static String farbeBlau = "#64A5BB";
    public final static String farbeSchwarz = "#232B2B";
    public final static String farbeGelb = "#FDD782";
    public final static String farbeWeiss = "#F7F9EF";
    public final static String farbeGrau = "#3B444B";
    // Rechteck für die Darstellung der Lebenspunkte vom Held
    public static Rechteck lebenspunkteHeld;

1.2 Spielbrett darstellen

Schreiben Sie eine Prozedur, die einen leeren Spielbrett mit den Lebenspunkten des Spielers darstellt. 

Gehen Sie dabei so vor: 

  • initailisieren Sie die Grafikkonsole 
  • erstellen sie einen Rechteck, der den leeren Spielbrett darstellen soll
Hinweis
  • stellen Sie die Lebenspunkte des Spielers dar 

Wir stellen Ihnen eine private Prozedur zur Verfügung, die die Lebenspunkte des Spielers darstellt. Sie müssen diese nur einfügen und an der richtigen Stelle aufrufen. 

  /**
    * Private Prozedur stellt Lebenspunkte vom Spieler dar
    * @param spieler - Spieler Held
    */
    private void stelleLebenspunkteDar(Spieler spieler) {
      // Variable für die Lebenspunkte des Spielers
        int aktuellePunkte = spieler.getLebenspunkte();
        // Lebenspunkte für die grafische Darstellung berechnen
        int punkte = (int)(aktuellePunkte * 848 / 150);
        // Lebenspunkte zeigen
      Text lebensanzeige = new Text(0, 475, "Lebenspunkte", Grafik.BLACK, 15, "");
        Rechteck platzLebenspunkte = new Rechteck(0, 500, 850, 20, "#006400", Grafik.BLACK, 1, "");
        lebenspunkteHeld = new Rechteck(1, 501, punkte, 18, Grafik.GREEN, Grafik.BLACK, 0, "");
    }
