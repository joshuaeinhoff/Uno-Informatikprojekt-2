GUI

Die Darstellung des Spiels wird durch die Klasse GUI ermöglicht. Für ein visuell ansprechendes Spiel ist es ratsam sich in vorhinein Gedanken zu einer Farbpallete zu machen. Die ausgewählten Hex-Codes können Sie, dann als Konstanten speichern.

Hints

Darstellung von einem leeren Canvas

Mittels dieser Methode wird eine leere weiße Fläche erstellt, die der Größe des späteren Spiels entspricht.

 

Hints

 

Wahl des Moduls

Hier geben Sie dem Spieler die Möglichkeit zwischen OOP1 und OOP2 zu wählen. Dies hat Einfluss auf die Fragen die der Spieler später beantworten muss. 

 

Hints

 

Wahl der Tempel

Erstellen Sie eine Funktion zur Darstellung der Tempelsymbol. Durch einen Klick auf das passende Symbol wird der Spieler zu einer Begrüßungsnachricht weitergeleitet. 
Wenn Sie möchten können sie, mit der waitFor() Methode, Animationen simulieren.

 

Darstellung der Begrüßungsnachricht

Die Begrüßungsnachricht gibt eine kurze Einführung für die folgende Quizfrage. 

Hints

Das Quiz darstellen

Nun fehlt noch die Quizfrage darzustellen. Der Methode soll eine Frage und Antwortmöglichkeiten übergeben werden und die ausgewählte Anwort wird zurückgegeben.

 

Das Spielfeld

Sie schreiben als erstes einige Methoden die Ihnen später bei dem zusammensetzen des Spielfelds helfen werden.

Darstellung der Kartenplätze (Optional?)

Diese Methode soll einen Hinweis auf den Platz der Kartenposition darstellen, das kann zum Beispiel durch eine kartenförmige Umrandung realisiert werden. 

Darstellung des Uno-Buttons

Erstellen Sie einen Uno-Button der erstmal noch keine Funktion hat und mit den Angaben von X- und Y- Position aufgerufen wird. 

Darstellung der Links-/Rechtsbuttons

Wir brauchen ebenfalls eine Methode die zwei Knöpfe der Links- und Rechtsverschiebung der Spielerhand darzustellen. Die Knöpfe sollten eine entsprechende Ausgabe auf die Konsole ausgeben. Die Methode wird wieder durch Angabe der X- und Y-Koordinate aufgerufen.

Darstellung des Kartenstapels

Fehlt nur noch die Darstellung von einem Kartenstapel von dem der Spieler Karten ziehen kann. Die Vorgehensweise ähnelt der der Darstellung der Platzhalterkarten. [Kartengröße z.B. (50,100)]

Erstellung der Spielfelddarstellung

Es ist ratsam für das Spielfeld eine andere Farbe als des Hintergrundes zu wählen und einen kleinen Rand zum Canvas zu lassen. 

Danach rufen Sie in passenden Abständen die Methode für die Platzhalterdarstellung, für der Karten der Händer der Spieler, auf. 
Für eine bessere Übersicht sollten Sie nicht alle Karten auf der Hand gleichzeitig anzeigen. (z.B. nur 10 von 20 möglichen Karten auf der Hand)
