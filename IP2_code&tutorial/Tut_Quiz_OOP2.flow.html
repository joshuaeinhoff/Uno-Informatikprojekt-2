Quiz: Einleitung für OOP2

Sie haben bis jetzt alle Klassen, die Sie für das UNO-Spiel brauchen, erstellt. Nun folgt das Quiz:

Das Quiz besteht aus insgesamt 50 Fragen: 20 zu OOP1-Themen und 30 zu OOP2-Themen. Der Spieler hat am Anfang die Möglichkeit sich zwischen OOP1 und OOP2 zu entscheiden. Danach muss der Spieler der Reihe nach die Tempeln durchqueren (Wasser-Tempel, Luft-Tempel, Erde-Tempel, Feuer-Tempel) und Quizfragen beantworten. In jedem Tempel gibt es fünf bis acht Fragen, von denen mindestens eine richtig beantwortet werden muss, um mit dem UNO-Spiel anfangen zu können.

Die Fragen für das Quiz werden von uns zur Verfügung gestellt. Sie befinden sich in verschiedenen Dateien, aus denen sie ausgelesen werden müssen. Das Lesen erfolgt duch sogenannte FilterReader-Klassen, die Sie noch nicht kennen und daher auch von uns zur Verfügung gestellt werden.

Tipp: Beachten Sie jede Klasse in ihr entsprechendes Package anzulegen!

3 Quizfrage

Um ein richtiges Quiz zu erstellen, braucht man selbstverständlich erstmal die Quizfragen. Dafür erstellen Sie jetzt eine eigene Klasse. Diese Klasse stellt eine Quizfrage mit den entsprechenden Eigenschaften im Spiel dar.

5.3 Quiz lösen 

Implementieren Sie die Methode quizGeloest(Spieler held, int quizNiveau, GUI gui) zum Beantworten der zu einem gegebenen Index im Fragenpool gespeicherten Frage.

1. Definieren Sie die lokalen Variablen:

  • antwortIstRichtig und setzen diese auf false
  • zufallsZahl, die den Index der Frage beschreibt
  • schaden für die Schadenspunkte und setzen diese auf 10

2. Wählen Sie je nach quizNiveau den Fragenpool zu OOP1 oder OOP2.

3. Wählen Sie eine Zufallsfrage aus dem Fragenpool aus und beantworten diese. Die Methode dazu haben Sie bereits implementiert. 

4. Falls eine Quizfrage falsch beantwortet wurde, müssen Sie dem Spieler Schadenspunkte hinzufügen.
